h1. Congrelate

Congrelate is a website and data aggregation framework written in Sinatra.

h3. Getting started

  * The following gems need to be installed: sinatra, fastercsv, httparty, open-uri, json, hpricot, htmlentities, technicalpickles-daywalker (via GitHub)
  * Copy config/database.yml.example to config/database.yml and modify as needed.
  * Copy sources/api_keys.yml.example to sources/api_keys.yml and add your API keys.
  * Run "rake db:migrate", then "rake sources:update". This may take a while.
  * Run "congrelate.rb" to start the server.
  * Visit "http://localhost:4567" in your browser.
